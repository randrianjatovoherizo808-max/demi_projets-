📖 Guide d'utilisation — PERMIS TIC

🎯 Présentation de la plateforme

PERMIS TIC est une plateforme de gestion de centre de formation en informatique. Elle comporte trois espaces : Public, Apprenant et Administrateur.

🌐 Espace Public (page d'accueil)

• Voir les formations : Cliquez sur les boutons Niveau A, B ou C pour parcourir les formations disponibles.

• S'inscrire : Cliquez sur "S'inscrire" ou sur le bouton "📝 S'inscrire" d'une formation. Remplissez le formulaire avec vos informations. Votre demande sera en attente de validation par l'admin.

• Statistiques : Le bandeau d'accueil affiche le nombre de formations, d'apprenants, de formateurs et le taux de réussite.

🎓 Espace Apprenant

• Connexion : Cliquez sur "Espace Apprenant" dans le menu. Entrez l'email et le mot de passe choisis lors de l'inscription.

⚠️ Votre compte doit être confirmé par l'administrateur pour pouvoir vous connecter.

• Mes Formations : Visualisez la ou les formations auxquelles vous êtes inscrit(e).

• Mes Leçons : Consultez les leçons disponibles pour vos formations.

• Mon Profil : Vos informations personnelles (nom, email, statut).

• Déconnexion : Cliquez sur "🚪 Déconnexion" dans la barre haute.

🔐 Espace Administrateur / Formateur

• Accès : Cliquez sur "Admin" dans le menu public. Identifiants par défaut : admin / admin123

• Dashboard : Vue d'ensemble avec statistiques et graphique des formations par niveau.

• Inscriptions : Valider ou supprimer les demandes d'inscription. Un clic sur "Confirmer" crée automatiquement le compte apprenant.

• Apprenants : Liste de tous les apprenants confirmés avec leur moyenne générale et statut (Admis / Ajourné).

• Formateurs : Ajouter et gérer les formateurs qui pourront se connecter à l'interface admin.

• Formations : Créer, modifier ou supprimer des formations (nom, durée, coefficient, places).

• Leçons : Ajouter des leçons associées à des formations. Les apprenants peuvent les consulter.

• Notes : Saisir les notes des apprenants par formation. La moyenne générale est calculée automatiquement.

• Coefficients : Modifier les coefficients de chaque formation pour le calcul de la moyenne pondérée.

• Paramètres : Modifier les informations du centre, changer le mot de passe admin, basculer entre mode clair et sombre.

💡 Astuces et conseils

• Les données sont sauvegardées automatiquement dans le navigateur (localStorage).

• Pour tester l'espace apprenant : créez une inscription → validez-la en admin → connectez-vous avec l'email et le mot de passe saisis.

• Mot de passe oublié : utilisez les liens "Mot de passe oublié" sur les formulaires de connexion.

• Le mode sombre est disponible dans Paramètres (admin) ou automatiquement selon vos préférences navigateur.

• Les niveaux A (Débutant), B (Intermédiaire) et C (Avancé) correspondent au Permis TIC officiel.

📌 Projet universitaire — Ce système est une simulation. Les données sont stockées localement dans votre navigateur et ne sont pas partagées entre appareils.
